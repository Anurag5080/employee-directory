# Assets Directory

This directory contains static assets for the Employee Directory application.

## Structure

- `images/` - Image files (logos, icons, etc.)
- `styles/` - Additional CSS files
- `fonts/` - Custom fonts

## Usage

Place all static assets in their respective subdirectories. These assets are typically imported in React components or included in the build process.

## Best Practices

- Optimize images for web use
- Use consistent naming conventions
- Group related assets in appropriate subdirectories
- Keep file sizes reasonable for better performance
